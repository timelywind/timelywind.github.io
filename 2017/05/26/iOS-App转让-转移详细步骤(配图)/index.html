iOS-App转让/转移详细步骤

发表于 2017-05-26

前言

当iOS的APP出于某种原因需要更换开发者账号的时候（出售等其他原因），无需下架删除APP，可以将其所有权转让给其他开发者账号。且在转让期间和转让后，APP评论和评分均可保留。转让后，用户还可以继续进行更新。

这里需要注意的是，集成过 iCloud， Passbook 或订阅类 iAP 将导致 App 无法转移到别的开发者帐号，即使之前集成过，新版禁用也不行。参考链接

下面是APP转让的具体步骤：

一、转出方操作

1. 登陆 iTunes Connect，点击我的应用>>app信息,点击最下面的转让app

2. 点继续

3. 填写Apple ID 与 Team ID

#####>> 下面可查看Team ID

#####4. 勾上，点击请求转让

#####>> 到此在转出账号下的操作就完成了

####二、接受者操作

#####1. 接受者登录接收App的Apple ID

#####2. 先点击Request

#####3. 打勾，服从协议

#####4. 再点击Review

#####5. 点击review，进入下面界面，填写接受者的基本信息，在这里，公司名称一经填写就无法改变，需要慎重

#####6. 填完之后，打勾，点击下面的Accept按钮

#####>>>>至此，app转移就此完成,一般需要三小时或者两天，有时候会很快，我转让的app就是两个小时就好了
